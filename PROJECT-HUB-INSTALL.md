# Add Orgasmaphoria to the Zigs Development Hub

1. Unzip the package.
2. Rename `orgasmaphoria-site` to `orgasmaphoria` if desired.
3. Copy it into the hub repository under `projects/`.
4. Add a project card on the Zigs Development homepage linking to `projects/orgasmaphoria/`.
5. Commit and push.

Suggested project-card details:

- **Name:** Orgasmaphoria
- **Category:** Artist website and member portal prototype
- **Description:** A multi-page music, community, document, event, store, account, message, privacy, accessibility, and staff publishing demonstration.
- **Status:** Client prototype
- **Technology:** HTML, CSS, JavaScript, localStorage, IndexedDB

Keep the repository private until the client is comfortable with the content, then publish only the approved static demonstration. Do not include real private member or staff documents in the public repository.
